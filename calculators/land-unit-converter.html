Indian Land Unit Converter

Bigha means different things in Bihar, UP, and Punjab. So does Katha. This converter knows all the regional variants — pick your units and convert instantly.

How to use ▼
  1. Two unit slots are active by default. Add up to 4 total using the + button.
  2. Use the region chips (North / South / East / West / Standard) to filter the dropdown to units from that region.
  3. Type any value in any box — all other boxes update automatically.
  4. Remove a slot with the ✕ button (minimum 2 required).
  5. Scroll down to the Reference Table for sq ft values, states, and notes for every unit.
State-specific units: Bigha, Katha, Dhur and others have very different values in different states. The dropdown lists them as e.g. Bigha (Bihar) vs Bigha (UP). Always match the state to your land's location.
